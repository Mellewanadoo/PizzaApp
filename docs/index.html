Pizzapp

API public officiel de Pizzapp-server.

GET /api/pizzas

Retourne la liste des pizzas.

POST /api/planets

Ajoute une nouvelle pizza.
Cet ajout est au format json et contient

  • name en string
  • price en number
  • size qui est "S" | "L" | "XL"
  • photo en string

PATCH /api/pizzas/id

Modifie une pizza.
Cette modification est au format json et contient

  • name en string
  • price en number
  • size qui est "S" | "L" | "XL"
  • photo en string

GET /api/planets

Retourne la liste des planètes.

POST /api/planets

Ajoute une nouvelle planète.
Cet ajout est au format json et contient

  • image en string
  • nom en string
  • localisation en string
  • paysage en string
  • especesInteligentes en string
  • premiereApparition en string

PATCH /api/planets/id

Modifie une planète.
Cette modification est au format json et contient

  • image en string
  • nom en string
  • localisation en string
  • paysage en string
  • especesInteligentes en string
  • premiereApparition en string

DELETE /api/planets/id

Supprime une planète.
